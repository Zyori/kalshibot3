# 2026 World Cup Logbook

The user's running scouting notes on teams across the 2026 World Cup — freeform
intuition that no feed provides (how a side defends, who fades late, keeper
tendencies, ref behavior). This is **the user's knowledge**, captured over time.

## How LUTZ uses this

- **Read it at session start**, with the other grounding docs. Before a read on a
  specific game, pull the notes for the two teams playing and factor them in.
- **Edit it only when the user asks** ("note that NZ defends deep", "log that…").
  Add the note under the right team's section as a dated bullet. Never rewrite or
  prune the user's notes unprompted, and never add notes the user didn't ask for.
- It is **scouting context, not a trade signal on its own** — it sharpens a read,
  it doesn't override live game state or the strategy docs.
- New team? Add a `## <Team>` section. Keep notes as short dated bullets.
